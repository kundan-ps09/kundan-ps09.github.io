AEM Installation

This tutorial describe how to install and start a AEM 6.0 Author and publisher instance.

How to install an Author instance:

  • Create a folder structure on your file system where you will store, install, and start AEM6.0 (e.g. C://Adobe/AEM6.0/author
  • Copy the AEM 6.0 quickstart JAR and license.properties file into the newly created folder structure
  • Rename the AEM quickstart JAR to aem6.0-author-p4502.jar
    - AEM6.0 = the application name
    - author = the WCM mode it will run in (e.g. author or publish)
    - 4502 = the port it will run in (e.g. any available port is acceptable)
    - If no port number is provided in the file name, AEM6.0 will select the first available port from the following list: (4502,8080,8081,8082,8083,8084,8085,8888,9362)
  • Double-click the aem6.0-author-p4502.jar
    - AEM will start as author on port 4502

Start AEM from command line

  • Once AEM is installed using jar file and started. Stop using "stop" button
  • From command line go to /crx-quickstart/bin
  • Run command start.bat

Change default JVM param

  • Open to /crx-quickstart/bin/start.bat in any editor of your choice
  • Search for "set CQ_JVM_OPTS"
  • Change JVM option based on your need

Stop AEM

  • If started AEM by double clicking jar file and just click on stop button in jar file
  • If started AEM from command line then run "stop" script from crx-quickstart/bin directory
